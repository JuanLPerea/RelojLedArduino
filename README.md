Reloj con matrices de LED y Arduino

Materiales:

1 Arduino Pro-mini
1 módulo RTC DS1302
4 matrices de led rojo de 8x8 con controlador MAX7219
1 matriz de 8x8 bicolor, controlada mediante 3 shift registers y un chip array darlington
1 modulo bluetooth
1 sensor DHT11 para medir temperatura y humedad relativa
1 electroimán controlado mediante un transistor Mosfet
3 botones
cables de conexión
1 placa para el circuito electrónico
